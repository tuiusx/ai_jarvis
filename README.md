# 🧠 AI Jarvis – Assistente de IA Local

AI Jarvis é um projeto de **assistente de Inteligência Artificial local**, offline-first, inspirado em sistemas como o Jarvis (Homem de Ferro), com foco em:

- Execução local (sem depender da internet)
- Arquitetura modular de agentes
- Visão computacional
- Automação inteligente
- Privacidade e controle total do usuário

## 🚀 Funcionalidades atuais

- Agente de IA com memória curta e longa
- Sistema de ferramentas (plugins)
- Gravação de vídeo automática
- Detecção de pessoas via visão computacional
- Execução totalmente offline
- Arquitetura preparada para GPU (NVIDIA)

## 🛠️ Tecnologias

- Python
- OpenCV
- Arquitetura de agentes
- Planejamento estruturado
- Design orientado a eventos

## 📂 Estrutura do projeto

ai_jarvis/
├── core/ # Agente, memória, planner, LLM
├── tools/ # Ferramentas (câmera, gravação, etc.)
├── interfaces/ # Interface de texto (futuro: voz, web)
├── recordings/ # Gravações de vídeo
└── main.py
